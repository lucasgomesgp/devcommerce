## :desktop_computer: Como usar
- É necesário criar sua API para fornecer os dados da aplicação e preencher como variável de ambiente, com a chave
```
    REACT_APP_URL=
```
-- Após isso é necessário rodar no terminal o comando abaixo:
```
    yarn start
```
